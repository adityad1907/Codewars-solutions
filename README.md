## 📖 About
This repository contains my solutions to various **Codewars** programming challenges (Katas), written across multiple languages as I learn them. It serves as a record of my problem-solving journey, documenting my progress as I strengthen my understanding of algorithms, data structures, and core programming concepts in **Python**, **C++**, and **Java**.

The repository is continuously updated as I complete new challenges, learn new languages, and improve existing solutions.

---

## 🎯 Objectives
- Practice problem-solving through daily coding challenges.
- Improve algorithmic thinking and logical reasoning.
- Strengthen fundamentals across Python, C++, and Java.
- Compare how the same problem is solved in different languages.
- Track progress across different Codewars difficulty levels.
- Build a consistent coding habit.
